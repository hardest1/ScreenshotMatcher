# ScreenshotMatcher

Screenshot matching tool based on A-Schmid/ScreenshotMatcher

# Required Libraries

OpenCV with contrib packages
gtk3
libappindicator3

# Usage

Start the daemon with:
```sh
./ScreenshotMatcher
```
